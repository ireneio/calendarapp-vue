# 預約表單系統 WEB APP

## User Story

- 用戶登入/登出 - Line Login
- 行事曆可以切換為週/日顯示
- 供萬年曆直接選擇日期
- 新增預約事件

## 架構

- SPA
- 前端：Vue2, Bootstrap4
- 後端：NodeJS, Express, MongoDB

### 工具

- Vue CLI 3
- Eslint Standard
